# Reyhoon Hotspot Login 🤦🏻‍

Nothing just a simple http client to log us in with less pain and effort.


## Usage

Set following environment variables and run the binary.

```bash
RHN_HS_IP
RHN_HS_USERNAME
RHN_HS_PASSWORD
```
